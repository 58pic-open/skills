绑定 API Key

安装完成后，绑定你的千图网 API Key 才能开始使用。只需配置一次，永久有效。

①
获取 API Key

访问千图 AI 开放平台，点击「新建令牌」创建你的专属 Key。

🔗 前往创建 API Key
🔑
API Key 格式：sk_xxxxxxxxxxxxxxxx
只显示一次 — 创建后请立即复制，关闭弹窗后将无法再次查看。
②
绑定到本地

在终端运行以下命令（把 sk_your_key 替换成你的真实 Key）：

bash
python3 ~/.claude/skills/58pic/scripts/init_config.py --api-key sk_your_key
③
完成！开始使用
🔒
你的 API Key 已加密存储在本地，不会传输到任何服务器。配置保存后永久有效，无需重复设置。

做同款

以千图素材或参考图为基础，AI 生成风格相近的新图片

⚠️
做同款必须提供参考图（千图 PID、图片 URL 或本地上传图片），与纯文字生图不同。
4 种使用方式
🖼
用千图 PID 做同款
最常用 · 以千图素材为参考
用 PID 12345678 做同款，换成蓝色调
Claude 自动获取该素材作为参考，生成风格相近但色调不同的新图
🌐
用图片 URL 做同款
参考任意在线图片
参考这张图 https://example.com/img.jpg，生成类似风格的圣诞海报
适合参考千图网以外的图片，直接粘贴链接即可
📤
上传本地图片做同款
直接上传你的参考图
（上传图片）基于这张图做同款，生成 4 张变体
在对话中上传图片，Claude 自动处理并传给 AI 模型
🔄
换模型重新生成
用不同模型探索更多变体
换一个模型再做同款，这次要卡通风格
Claude 会列出其他可用模型，选择后立即重新生成

参数文档

搜索、生成、下载的完整参数说明

🔍
素材搜索参数
参数说明自然语言示例
keyword必填 搜索关键词 「春节海报」「商务人物」
page可选 页码，默认第 1 页，最多 100 页 「第 2 页」「翻下一页」
did可选 一级分类 ID，0或不传=全部。可选值：2=海报展板 / 3=电商淘宝 / 4=装饰装修 / 5=网页UI / 6=音乐音效 / 7=3D素材 / 8=PPT模板 / 10=背景 / 11=免抠元素 / 12=Excel模板 / 14=简历模板 / 15=Word模板 / 16=社交媒体 / 17=插画 / 40=字库 / 41=艺术字 / 53=高清图片 / 56=视频模板 / 57=元素世界 / 60=AI数字艺术 / 66=品牌广告 「只搜插画类」「找免抠元素」「搜 PPT 模板」
ai-search可选 启用 AI 向量搜索，适合描述性长句查询 「用 AI 搜索 红色喜庆灯笼祥云」
🎨
AI 生图参数
参数说明自然语言示例
prompt / ai_title可选 图片内容描述；省略时默认「开放平台同款」 「红色喜庆春节贺卡，灯笼烟花」
model必填 模型 ID（整数或字符串），从 available-models 接口获取 「用插画风格模型」
generate_nums可选 生成数量，默认 1 张，范围 1–16 「生成 4 张」
reference_image_url三选一 单张参考图 URL（做同款必须提供参考图） 「参考 https://...」
reference_image_urls三选一 多张参考图 URL 列表 「同时参考这两张图」
image_base64三选一 本地图片 Base64 编码（≤8MB），上传本地图片时使用 （上传本地图片）「基于这张图做同款」
picid可选 千图素材 PID，用于同款逻辑（配合 reference_image_url） 「用 PID 12345 做同款」
media_type必填 固定传 "image"，传其他值返回 400 （自动填写，无需手动指定）
⚙️
配置与工具命令
常用命令
# 绑定 API Key $ python3 init_config.py --api-key sk_your_key # 查看当前配置（脱敏显示） $ python3 init_config.py --show # 查看可用 AI 模型列表 $ python3 list_models.py # 设置默认模型（省去每次选择） $ python3 init_config.py --default-model MODEL_ID # 重置所有配置 $ python3 init_config.py --reset
📁
配置文件默认保存在 ~/.58pic_config.json，仅当前用户可读。请不要分享给他人。
