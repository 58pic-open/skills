Claude Code / Cowork
Anthropic 官方 AI 工具 · Skills 原生支持
①
选择安装方式

推荐：直接从 npm 安装（最新版本）：

bash
npx skills add @58pic/skill

也可从 GitHub 安装：

bash
npx skills add github:58pic-open/skills#path=58pic

下载 zip 解压后，指定本地目录路径安装：

bash
# 1. 下载并解压
curl -L https://preview.58pic.com/AI/SKILLS/58pic.zip -o 58pic.zip
unzip 58pic.zip -d ./58pic

# 2. 本地路径安装（目录需含 SKILL.md + package.json）
npx skills add ./58pic
💡本地目录需包含 SKILL.md 和 package.json，工具会自动读取 skill 元数据。
②
验证安装
bash
npx skills list

3
配置 API Key

安装完成后，需绑定千图网 API Key 才能开始使用。

① 获取 API Key

访问千图 AI 开放平台，点击「新建令牌」创建：

🔗 前往创建 API Key
🔑
API Key 格式：sk_xxxxxxxxxxxxxxxx
仅在创建时显示一次，请立即复制保存！
② 绑定 API Key

在终端运行（将 sk_your_key 替换为实际 Key）：

bash
python3 ~/.claude/skills/58pic/scripts/init_config.py --api-key sk_your_key
③ 完成！开始使用
✅
配置保存后永久有效，Key 以加密方式存储在本地，不会上传到任何服务器。

🎭 做同款

以千图素材或参考图为基础，AI 生成风格相近的新图片

⚠️
做同款接口必须提供参考图（千图 PID、图片 URL 或本地图片），与纯文字生图不同。
典型使用场景
🖼
用千图 PID 做同款
最常用方式，以千图素材为参考
用 PID 12345678 做同款，换成蓝色调
Claude 会自动获取该素材图片作为参考，生成风格相近但色调不同的新图
🌐
用图片 URL 做同款
直接使用在线图片链接
参考这张图 https://example.com/img.jpg，生成一张类似风格的圣诞海报
适合参考非千图网的图片，直接粘贴 URL 即可
📤
上传本地图片做同款
直接上传本地参考图
（上传图片）基于这张图做同款，生成 4 张变体
在对话中上传图片，Claude 会自动将其转换为 Base64 进行处理
🔄
换模型重新生成
使用不同模型生成更多变体
换一个模型再做同款，这次要卡通风格
Claude 会列出其他可用模型，选择后重新生成

📖 命令参考

搜索、生成、下载的完整参数说明

🔍
素材搜索参数
参数说明示例
keyword搜索关键词（必填）「春节海报」「商务人物」
page 可选页码，默认第 1 页，最多 100 页「第 2 页」「翻到下一页」
kid 可选分类筛选，0=全部
130=免抠 · 275=广告 · 735=插画
668=摄影 · 743=GIF · 276=字体		「只搜插画类」「找免抠元素」
ai-search 可选启用 AI 向量搜索，适合描述性查询「用 AI 搜索 红色喜庆灯笼祥云」
🤖
AI 生成参数
参数说明示例
prompt图片描述（必填）「红色喜庆春节贺卡 灯笼烟花」
model 可选模型 ID，首次使用会列出可选项「用插画风格模型」
nums 可选生成数量，默认 2 张「生成 4 张」
ref-pid 可选参考素材 PID（做同款）「用 PID 12345 做同款」
ref-url 可选参考图片 URL（做同款）「参考 https://...」
⚙️
配置与工具命令
常用命令
# 设置 API Key $ python3 init_config.py --api-key sk_your_key # 查看当前配置（脱敏显示） $ python3 init_config.py --show # 查看可用 AI 模型列表 $ python3 list_models.py # 设置默认模型 $ python3 init_config.py --default-model MODEL_ID # 重置所有配置 $ python3 init_config.py --reset
📁
配置文件默认保存在 ~/.58pic_config.json，仅当前用户可读，请勿分享给他人。
